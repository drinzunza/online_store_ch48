## GIT

# initialize repo
git init

# add .gitignore


# add a remote (github url)
git add remote origin <REPO URL>


# add files (stage changes)
git add .


# create a commit  (snapshot)
git commit -m "ANY MESSAGE"


# push the first time
git push -u origin main     ( or master )

# push nth time
git push
